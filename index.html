Projects

bfetch

bfetch is a simple system information too for UNIX systems written in Bash. It supports various display informations like the uptime, the kernel and OS version, the disk usage and the amount of installed packages.

LibreTube

LibreTube is an alternate YouTube frontend with a modern UI. Its focus is on privacy, therefore the app doesn't send any data to Google by using Piped instances. That's the reason why users can still subscribe and create their own playlists.

LibreTube Website

The LibreTube website is a modern, from-scratch-written webpage using HTML, CSS and JavaScript only. It showcases LibreTube and all its various features.
