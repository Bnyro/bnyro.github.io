Projects

bfetch

bfetch is a simple system information tool for UNIX-based systems written in Bash. It supports various display informations like the uptime, the kernel and OS version, the disk usage and the amount of installed packages.

LibreTube

LibreTube is an alternative YouTube frontend with a modern UI. Its focus is on privacy, therefore the app doesn't send any data to Google by using Piped instances. The app has support for SponsorBlock as well as many other advanced features.

LibreTube Website

The LibreTube website is a modern, from-scratch-written webpage using HTML, CSS and JavaScript only. It showcases LibreTube and all its various features as well as providing an FAQ.
